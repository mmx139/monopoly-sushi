# 复盘：骰子不显示数字 B00016

## 问题描述
按下投骰子后，骰子不显示投出的数字，仍显示 "?" 或旧值。

## 时间线
| 时间 | 事件 |
|---|---|
| 2026-03-31 晚间 | 用户反馈骰子投出后不显示数字 |
| 2026-03-31 晚间 | 分析 Dice.vue 和 App.vue 代码 |
| 2026-03-31 晚间 | 发现问题：Dice 组件动画时 value 未设置 |
| 2026-03-31 晚间 | 修改：App.vue 先调用 rollDice() 设置值，Dice 组件 watch 变化触发动画 |

## 根因
- **表面原因**：Dice 组件在动画期间显示的是旧值
- **真正原因**：
  - 原流程：点击 → 动画 → emit('roll') → App 处理
  - 但 App 处理后 value 未立即传递给 Dice
  - Vue 响应式更新时机问题

## 解决方案
1. Dice.vue：watch props.value 变化时播放动画
2. App.vue：onRoll 时先调用 rollDice() 设置值，再用 setTimeout 800ms 后调用 movePlayer()

## 教训
1. **父子组件数据流**：props 变化需要显式 watch，不能依赖 computed
2. **动画时序问题**：投骰子动画 800ms，期间不能再次触发

## 预防措施
- 涉及动画和状态更新的组件，添加 console.log 调试
- 考虑使用 Vue DevTools 检查响应式数据流
