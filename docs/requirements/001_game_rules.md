# Monopoly-Sushi 游戏需求文档

## 游戏概述

**名称**: 一蓑烟雨任平生 (Monopoly-Sushi)
**类型**: 大富翁类桌游电子版
**主题**: 苏轼生平与诗词文化
**玩家数**: 2-4 人（支持 AI）

---

## 游戏元素

### 2.1 角色

| 角色 | 描述 |
|------|------|
| 苏轼 | 主角 |
| 苏辙 | 苏轼之弟 |
| 王安石 | 变法派人物 |
| 张怀民 | 苏轼好友 |

### 2.2 棋盘

- **规格**: 正方形，72 格
- **边角**: 4 格（起始点等特殊位置）
- **每边**: 17 格（不含边角）
- **可购地皮**: 32 格（一边 8 个）

### 2.3 货币系统

| 类型 | 面额 |
|------|------|
| 铜币 | 10 / 50 / 100 |
| 银币 | 200 / 500 |
| 金币 | 1000 / 5000 |

**初始资金**: 15000

**初始配置**:
- 10/50/100 铜币各 5 枚
- 200 银币 6 枚
- 500 银币 2 枚
- 1000/5000 金币各 2 枚

### 2.4 骰子

- 共 2 枚
- 每回合投 1 枚，按点数前进
- 起始：投 2 枚，点数最大者先手，之后按顺时针轮流

## 卡牌系统

### 4.1 问答卡

**类型**: 诗词/文学知识（关于苏轼）

**规则**:
- 回答正确按当前地皮要求奖励
- 当前地皮无要求：正确 +100
- 错误：无奖励无惩罚
- 使用后不放回

**题库示例**:
- 苏轼是哪个朝代的文学家？ → 宋朝/北宋
- 苏轼字什么？ → 子瞻/和仲
- 苏轼号什么？ → 铁冠道人/东坡居士
- "但愿人长久，千里共婵娟" 出自哪首诗？ → 《水调歌头·明月几时有》

### 4.2 诗词题卡

**规则**:
- 准确背诵一句诗词: +100
- 错误背诵: -200
- 背出全诗: 赐予当前手头货币的一半
- 使用后不放回

**包含 9 首诗词**:
1. 《江城子·乙卯正月二十日夜记梦》
2. 《念奴娇·赤壁怀古》
3. 《水调歌头·明月几时有》
4. 《饮湖上初晴后雨》
5. 《蝶恋花·春景》
6. 《卜算子·黄州定慧院寓居作》
7. 《和子由渑池怀旧》
8. 《题西林壁》
9. 《惠崇春江晚景》

## 特殊地皮

### 5.1 类型

- **诗词事件**: 触发诗词题
- **问答事件**: 触发问答卡
- **奖励事件**: 抽取奖励道具
- **入土为安**: 获得墓碑
- **兄弟团聚**: 获得花束

---

## 胜利条件

- 场上只剩一位未破产的玩家
- 破产判定：金钱为负且无法出售资产

---

## 游戏流程

1. **开局**: 各玩家投骰子决定先手
2. **回合**: 投骰子 → 移动 → 触发地皮事件
3. **回合结束**: 判定是否破产 → 下一玩家
4. **胜利**: 只剩一位玩家时结束
