# Monopoly-Sushi 游戏需求文档

## 游戏概述

**名称**: 一蓑烟雨任平生 (Monopoly-Sushi)
**类型**: 大富翁类桌游电子版
**主题**: 苏轼生平与诗词文化
**玩家数**: 2-4 人（支持 AI）

---

## 游戏元素

### 2.1 角色

| 角色 | 描述 |
|------|------|
| 苏轼 | 主角 |
| 苏辙 | 苏轼之弟 |
| 王安石 | 变法派人物 |
| 张怀民 | 苏轼好友 |

### 2.2 棋盘

- **规格**: 正方形，72 格
- **边角**: 4 格（起始点等特殊位置）
- **每边**: 17 格（不含边角）
- **可购地皮**: 32 格（一边 8 个）

### 2.3 货币系统

| 类型 | 面额 |
|------|------|
| 铜币 | 10 / 50 / 100 |
| 银币 | 200 / 500 |
| 金币 | 1000 / 5000 |

**初始资金**: 15000

**初始配置**:
- 10/50/100 铜币各 5 枚
- 200 银币 6 枚
- 500 银币 2 枚
- 1000/5000 金币各 2 枚

### 2.4 骰子与回合

- 共 2 枚
- 每回合投 1 枚，按点数前进，到达格子后触发格子的效果，完毕后回合结束，轮到下一玩家投
- 起始：投 2 枚，按点数从大到小排列后续顺序，点数相同的玩家按投出的先后排序

---

## 地皮系统

### 3.1 可购地皮：浅绿底色

#### 3.1.1 购买/升级
- 踩中空的可购地皮可以选择是否购买并摆放一级房屋
- 踩中自己的地皮可以选择是否升级
- 踩中自己的地皮可以获得奖金
- 踩中他人的地皮则根据该地皮的等级对应的过路费付费给地皮拥有者

#### 3.1.2 房屋等级

| 等级 | 造价 | 过路费 | 升级费 |
|------|------|--------|--------|
| 一级 | 1000 | 500 | - |
| 二级 | - | 800 | 800 |
| 三级 | - | 1000 | 800 |

#### 3.1.3 奖金
- 一级: 100
- 二级: 150
- 三级: 200

### 3.1.4 玩家可选定自己的一处地皮拍卖
拍卖时，4名玩家轮流出价或Pass，只有当任何3名连续玩家全部pass，则上一个出家玩家按出价买下这块地皮

| 等级 | 起拍价 |
|------|--------|
| 一级房屋 | 500 |
| 二级房屋 | 1000 |
| 三级房屋 | 1500 |

### 3.2 特殊地皮

#### 3.2.1 问答事件：浅红底色
- 抽一张问答题卡
- 当前地皮有具体要求（参照地图需求文档中对应格子的"效果"）：回答正确奖励按照当前地皮具体要求
- 当前地皮无具体要求：回答正确获得100
- 回答错误无奖励/惩罚

#### 3.2.2 特殊事件：浅紫底色
参照地图需求文档中对应格子的"效果"

#### 3.2.3 诗词事件：潜橙底色
- 若地图需求文档中对应格子的“效果”有内容，则强制执行该效果
- 若地图需求文档中对应格子的“效果”无内容，则让玩家选择：
    1. 抽一张诗词题卡
    2. 背诵（这里列出当前格子的“名称”，即为指定诗词名）
- 答题效果
    1. 准确写出一句完整诗词（需判断符合诗词的一个断句）: +100
    2. 写出的一句诗词不符合一句完整诗词：-200
    3. 准确写出全诗（忽略标点符号，允许错3个字）: 赐予玩家当前手头货币的一半

#### 3.2.4 奖励事件：浅蓝底色
从可以通过“奖励事件“获得的道具卡片中随机抽取一张

#### 3.2.5 惩罚事件：浅灰底色
抽一张惩罚卡片

#### 3.2.6 随机事件：浅黄底色
投一次骰子，单数抽取一张惩罚卡片，双数抽取一张道具卡片

## 卡牌系统

### 4.1 问答卡
- 题库中每题一张卡，游戏起始随机排序
- 使用后，排回问答卡堆最后

**题库**:   “/”分割的答案，答中任何一个就算对
- 苏轼是哪个朝代的文学家？ → 宋朝/北宋
- 苏轼字什么？ → 子瞻/和仲
- 苏轼号什么？ → 铁冠道人/东坡居士
- “三苏”指哪三人？ → 苏洵，苏轼，苏辙
- “苏堤”与其他哪两堤并称“西湖三堤”？ → 白堤，杨公堤
- "但愿人长久，千里共婵娟" 出自哪首诗？ → 水调歌头明月几时有/水调歌头
- “大江东去，浪淘尽，千古风流人物”这句词描写的是哪一个古代战场？ → 赤壁
- “宋四家”的“苏黄米蔡”中的“黄”是指？ → 黄庭坚
- 苏轼被认为是宋代词坛哪一个流派的开创者？ → 豪放派 
- “横看成岭侧成峰，远近高低各不同”描写的是哪座山？ → 庐山
- 哪道著名的传统菜肴是由苏轼在黄州时改良并推广的？ → 东坡肉
- 苏轼与谁并称"苏辛" → 辛弃疾
- 苏轼在《记承天寺夜游》中，与哪位好友一起在月色下散步？ → 张怀民
- 《念奴娇·赤壁怀古》中提到的“羽扇纶巾”形容的是谁？ → 周瑜
- 苏轼的哪位学生位列“宋四家”？ → 黄庭坚
- “老夫聊发少年狂，左牵黄，右擎苍”，“苍”是指什么动物？ → 苍鹰
- 《六月二十七日望湖楼醉书》中“白雨跳珠乱入船”下一句是？ → 卷地风来忽吹散
- 苏轼曾提到“宁可食无什么，不可居无什么”？ → 肉，竹
- 《和子由渑池怀旧》是写给谁的？ → 苏辙
 

### 4.2 诗词题卡
- 题库中每题一张卡，游戏起始随机排序
- 使用后，排回诗词卡堆最后

**包含 9 首诗词**:
1. 《江城子·乙卯正月二十日夜记梦》
十年生死两茫茫，不思量，自难忘。
千里孤坟，无处话凄凉。
纵使相逢应不识，尘满面，鬓如霜。
夜来幽梦忽还乡，小轩窗，正梳妆。
相顾无言，惟有泪千行。
料得年年肠断处，明月夜，短松冈。

2. 《念奴娇·赤壁怀古》
大江东去，浪淘尽，千古风流人物。
故垒西边，人道是，三国周郎赤壁。
乱石穿空，惊涛拍岸，卷起千堆雪。
遥想公瑾当年，小乔初嫁了，雄姿英发。
羽扇纶巾，谈笑间，樯橹灰飞烟灭。
故国神游，多情应笑我，早生华发。
人生如梦，一尊还酹江月。
江山如画，一时多少豪杰。

3. 《水调歌头·明月几时有》
明月几时有？把酒问青天。不知天上宫阙，今夕是何年。我欲乘风归去，又恐琼楼玉宇，高处不胜寒。起舞弄清影，何似在人间。
转朱阁，低绮户，照无眠。不应有恨，何事长向别时圆？人有悲欢离合，月有阴晴圆缺，此事古难全。但愿人长久，千里共婵娟。

4. 《饮湖上初晴后雨》
水光潋滟晴方好，山色空蒙雨亦奇。
欲把西湖比西子，淡妆浓抹总相宜。

5. 《蝶恋花·春景》
花褪残红青杏小。燕子飞时，绿水人家绕。枝上柳绵吹又少，天涯何处无芳草！
墙里秋千墙外道。墙外行人，墙里佳人笑。笑渐不闻声渐悄，多情却被无情恼。

6. 《卜算子·黄州定慧院寓居作》
缺月挂疏桐，漏断人初静。
谁见幽人独往来，缥缈孤鸿影。
惊起却回头，有恨无人省。
拣尽寒枝不肯栖，寂寞沙洲冷。

7. 《和子由渑池怀旧》
人生到处知何似，应似飞鸿踏雪泥。
泥上偶然留指爪，鸿飞那复计东西。
老僧已死成新塔，坏壁无由见旧题。
往日崎岖还记否，路长人困蹇驴嘶。

8. 《题西林壁》
横看成岭侧成峰，远近高低各不同。
不识庐山真面目，只缘身在此山中。

9. 《惠崇春江晚景》
竹外桃花三两枝，春江水暖鸭先知。
蒌蒿满地芦芽短，正是河豚欲上时。

## 胜利条件

- 场上只剩一位未破产的玩家
- 破产判定：金钱为负且无法出售资产

---

## 游戏流程

1. **开局**: 各玩家投骰子决定先手
2. **回合**: 投骰子 → 移动 → 触发地皮事件
3. **回合结束**: 判定是否破产 → 下一玩家
4. **胜利**: 只剩一位玩家时结束
