## 素材使用策略

### 第一阶段（MVP）

1. **棋盘**: 使用 `image61.png` 作为棋盘背景
2. **玩家立牌**: 使用 `image2.png` 裁剪出4个角色
3. **道具图标**: 使用 `image6.png`, `image7.png` 等
4. **骰子**: 使用 `image12.png`

### 第二阶段（优化）

1. 根据 `image53-60.png` 校对地图文字描述
2. 重新绘制更精细的棋盘（SVG矢量）
3. 中国风大背景使用 `image1.png` 或自行绘制
