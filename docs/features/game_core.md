# Monopoly-Sushi 功能规格

## 概述

本文档描述"一蓑烟雨任平生"游戏的功能规格。

---

## F1: 游戏房间系统

### F1.1 创建房间

- 玩家可以创建游戏房间
- 房主获得房间 ID
- 房间最大 4 人

### F1.2 加入房间

- 其他玩家通过房间 ID 加入
- 玩家选择角色（苏轼/苏辙/王安石/张怀民）
- 等待所有人准备后开始

### F1.3 房间状态

| 状态 | 描述 |
|------|------|
| waiting | 等待玩家加入 |
| playing | 游戏进行中 |
| finished | 游戏结束 |

---

## F2: 游戏流程

### F2.1 游戏开始

1. 所有玩家投 2 枚骰子决定先手
2. 点数最大者先手
3. 之后按顺时针顺序轮流

### F2.2 回合流程

```
回合开始
  → 投骰子
  → 移动（按点数前进）
  → 触发地皮效果
  → 执行操作（购买/升级/使用道具）
  → 回合结束
  → 下一玩家
```

### F2.3 胜利条件

- 场上只剩一位未破产玩家
- 破产：金钱为负且无法出售资产

---

## F3: 骰子系统

### F3.1 骰子规则

- 每回合投 1 枚骰子（1-6点）
- 点数决定移动格数
- 起始时投 2 枚决定先手

### F3.2 道具效果

| 道具 | 效果 |
|------|------|
| 马车 | 可再投 1 枚骰子 |
| 如意令 | 指定移动 1-6 格 |

---

## F4: 地皮系统

### F4.1 可购地皮

- 共 32 块可购地皮
- 踩中空地皮可选择购买
- 地皮价格：1000

### F4.2 房屋升级

| 等级 | 升级费用 | 过路费 | 奖金 |
|------|----------|--------|------|
| 一级 | 1000 | 500 | 100 |
| 二级 | 800 | 800 | 150 |
| 三级 | 800 | 1000 | 200 |

### F4.3 过路费

- 踩中他人地皮需缴过路费
- 房屋等级越高，过路费越高

### F4.4 拍卖规则

- 可将自己的房屋拍卖
- 起拍价：一级500 / 二级1000 / 三级1500

---

## F5: 货币系统

### F5.1 初始资金

- 每位玩家初始 15000

### F5.2 货币类型

| 类型 | 面额 |
|------|------|
| 铜币 | 10 / 50 / 100 |
| 银币 | 200 / 500 |
| 金币 | 1000 / 5000 |

## F6: 卡牌系统

### F6.1 问答卡

- 共 15 题（苏轼相关文学知识）
- 正确按地皮要求奖励或+100
- 错误无惩罚
- 使用后不放回

### F6.2 诗词题卡

- 共 9 首诗词
- 背诵正确 +100
- 背诵错误 -200
- 背出全诗得一半货币
- 使用后不放回

## F8: AI 玩家

### F8.1 决策规则

1. **购买决策**: 有足够资金时优先购买高价值地皮
2. **升级决策**: 优先升级已形成组合的地皮
3. **道具使用**: 根据局势决定是否使用道具
4. **投降决策**: 资金不足时考虑投降

### F8.2 估价函数

- 地皮价值 = 基础价格 + 房屋升级投入
- 考虑位置因素（是否在要道）

## F9: 网络同步

### F9.1 WebSocket 消息

| 消息类型 | 描述 |
|----------|------|
| player_joined | 玩家加入 |
| player_left | 玩家离开 |
| game_start | 游戏开始 |
| turn_start | 回合开始 |
| dice_rolled | 骰子结果 |
| player_moved | 玩家移动 |
| property_purchased | 地皮购买 |
| card_drawn | 抽取卡牌 |
| game_over | 游戏结束 |

### F9.2 状态同步

- 房主作为权威游戏状态
- 所有操作通过房主广播
